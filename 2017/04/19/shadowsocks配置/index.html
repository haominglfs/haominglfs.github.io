#shadowsocks配置

  • 第一步 用远程工具登录aws主机

  • 第二步：安装shadowsocks依赖

    1. sudo -s //获取超级管理员权限

    2. apt-get update//更新apt-get

    3. apt-get install python-pip//安装pyton包管理工具

    4. pip install shadowsocks//安装shadowsocks

    5. ssserver -c /etc/shadowsocks.json -d start//启动shadowsocks

  • 第三步:配置shadowsocks

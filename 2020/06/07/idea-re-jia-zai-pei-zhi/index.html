热加载方式

  1. JRebel 收费，有点贵

DCEVM+Hotswap Agent

下载DCEVM

latest release of DCEVM Java patch

sudo java -jar DCEVM-8u181-installer-build2.jar

选择jdk安装目录，并点击”Install DCEVM as altjvm”按钮

下载Hotswap agent jar

latest release of Hotswap agent jar

放到任意目录

安装idea插件 HotSwapAgent

配置插件

以debug模式启动应用程序
